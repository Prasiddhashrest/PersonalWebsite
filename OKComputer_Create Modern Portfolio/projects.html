SustainabilityScoring.swift 
class SustainabilityScorer {
    func calculateProductScore(product: Product) -> Double {
        // Distance-based scoring for food miles
        let distanceScore = calculateDistanceScore(miles: product.foodMiles)
        
        // Category-based sustainability weighting
        let categoryMultiplier = getCategoryMultiplier(category: product.category)
        
        // Combined sustainability score
        let baseScore = distanceScore * categoryMultiplier
        
        return min(max(baseScore, 0.0), 100.0)
    }
    
    private func calculateDistanceScore(miles: Double) -> Double {
        // Exponential decay based on distance
        return 100.0 * exp(-miles / 1000.0)
    }
}

  data = water_data

# Perform kriging interpolation
krige_model <- krige(
  formula = water_quality ~ 1,
  locations = water_points,
  newdata = grid_data
)

# Visualize results in Shiny app
output$map <- renderLeaflet({
  leaflet() %>%
    addTiles() %>%
    addRasterImage(krige_model, colors = "viridis") %>%

Comprehensive empirical investigation into the Kuznets Curve hypothesis, analyzing the nonlinear relationship between GDP per capita and income inequality using advanced econometric techniques and a comprehensive 145-country panel dataset.

Research Methodology

  • Panel data econometrics with fixed and random effects models
  • Nonlinear regression analysis and polynomial specifications
  • Robustness testing with alternative specifications
  • Cross-sectional and temporal analysis

Technology Stack

Stata R Python Econometrics Panel Data Statistical Analysis
kuznets_analysis.do 
* Kuznets Curve Analysis - Stata Code
clear all
use "harvard_dataverse_panel.dta", clear

* Set panel structure
xtset country_id year

* Polynomial specification for Kuznets Curve
generate gdp_sq = gdp_per_capita^2

* Fixed effects regression
xtreg gini_coefficient gdp_per_capita gdp_sq control_vars, fe robust

* Test for inverted U-shape
test gdp_per_capita = -gdp_sq

* Marginal effects analysis
margins, dydx(gdp_per_capita) atmeans

* Robustness checks
xtreg gini_coefficient gdp_per_capita gdp_sq control_vars, re robust

Key Findings

Inverted U-Curve Confirmation

Confirmed the Kuznets Curve hypothesis with statistically significant results across multiple model specifications

Policy Implications

Findings suggest that economic development initially increases inequality but eventually leads to more equitable distributions

Cross-Country Variations

Identified significant variations in the relationship across different regions and development stages
